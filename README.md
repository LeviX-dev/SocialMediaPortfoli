## Running the Project

Clone the repository

```
git clone https://github.com/yourusername/portfolio.git
```

Navigate into the project directory

```
cd portfolio
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

Open your browser and visit

```
http://localhost:5173
```
